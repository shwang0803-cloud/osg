# 🧠 OSD: Open Source Design Project
> AI 기반 결제 카드 디자인 생성 시스템  
> 사용자의 요구를 바탕으로 AI가 자동으로 카드 이미지를 생성하는 오픈소스 프로젝트입니다.

---

## 📘 프로젝트 개요
이 프로젝트는 사용자가 채팅을 통해 원하는 카드 디자인을 입력하면,  
AI가 해당 요구를 분석하여 결제 카드 이미지로 시각화해주는 시스템입니다.  
카드스킨 형태로 인쇄하거나, 카드사의 커스터마이징 기능(예: 토스, 현대카드)을 통해  
실제 결제용 카드로 구현할 수 있도록 설계되었습니다.  

---

## 🚀 주요 기능
- 🗨️ **대화형 AI 인터페이스**: 사용자가 자연어로 디자인 요구 입력  
- 🎨 **이미지 생성 엔진**: 회색/검정 조합 등 색상, 패턴, 폰트 반영  
- 💾 **결과 저장 기능**: PNG / PDF 파일 형태로 내보내기  
- 🧩 **디자인 구조화**: JSON 기반 스키마로 카드 규격(85.6×53.98mm) 자동 조정  
- 🔁 **수정/피드백 반영**: “폰트 더 굵게” 등 문장형 수정 요청 반영
